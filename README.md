# Tourism Experience Analytics

## 📌 Overview
This project focuses on building a Machine Learning-based system for tourism data analysis, prediction, and recommendation.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Classification & Regression Models
- Recommendation System
- Streamlit Deployment

## 🛠 Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, SQL, Streamlit

## 📊 Outcome
Developed a predictive and recommendation system to enhance tourism experience using data-driven insights.
